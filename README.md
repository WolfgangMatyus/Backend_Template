# Backend_Template
Server mit Node.js, Express und Postgres Datenbank

## start:
    npm server.js 

## Install:
node.js

postgreSQL

#### Express
    npm install express
#### Body-parser
    npm install body-parser
#### Postgres node.js package 
    npm install pg
#### jest und Suptertest
    npm install --save-dev jest supertest
#### change restart mit nodemon
    npm install -g nodemon
